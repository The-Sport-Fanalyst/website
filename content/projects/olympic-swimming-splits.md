## Overview

Split data is the raw material for swimming analytics, but it's scattered across PDFs and results books. This project standardizes it into tidy CSVs: one row per length, per swimmer, per race.

## Coverage

All Olympic finals 2000–2024 and a growing share of Paralympic finals. Each record carries reaction time, cumulative time, and split time, plus turn margin when the source supports it.

## Documentation

Every column is documented in the data dictionary, and every source is cited per-race so analyses can trace each number back to its origin.
