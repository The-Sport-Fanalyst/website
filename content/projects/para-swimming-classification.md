## Overview

Para swimming's classification system is essential context for any Paralympic analysis, but it's poorly understood outside the sport. This project explains it visually and tests how balanced each class is.

## What's inside

An interactive class-to-event map, distribution plots of finishing-time spread within each class, and a plain-language guide for newcomers — because trustworthy analysis starts with understanding the rules.
