## Question

Conventional wisdom says judges "save room" early in a rotation, so later competitors score higher. We test that claim rigorously.

## Approach

We model execution score against start position with athlete and apparatus fixed effects, controlling for difficulty. The write-up reports effect sizes with confidence intervals and shares the full analysis notebook so anyone can rerun it.
